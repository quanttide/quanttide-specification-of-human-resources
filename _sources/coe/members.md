# 成员

**成员(`members`)** 

字段定义如下：

- **ID(`id`)**：系统定义的唯一标识符。
- **名称(`name`)**：成员的姓名。
- **职务(`position`)**：成员在COE中的职务。
- **职级(`level`)**：成员的职级。
- **加入日期(`join_date`)**：成员加入COE的日期。
- **联系方式(`contact_info`)**：成员的联系方式。
- **状态(`status`)**：成员的当前状态，如“活跃”、“离职”等。
