# 职责

**职责(`responsibilities`)**

字段定义如下：

- **ID(`id`)**：系统定义的唯一标识符。
- **描述(`description`)**：职责的详细描述。
- **优先级(`priority`)**：职责的优先级，如“高”、“中”、“低”。
- **截止日期(`due_date`)**：职责的截止日期。
- **状态(`status`)**：职责的当前状态，如“未开始”、“进行中”、“完成”等。
